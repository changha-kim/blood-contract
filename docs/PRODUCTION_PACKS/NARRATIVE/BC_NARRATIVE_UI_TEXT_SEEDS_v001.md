PoC에서 필요한 최소 UI 텍스트 씨앗. **짧고, 행동을 설명하고, 반복에 견디는 문장**만 둔다.

## 상태/전투(Commit Combat)
- TELEGRAPH(예고):
  - "예고"
  - "의도 드러남"
- COMMIT/LOCK(확정):
  - "확정(LOCK)"
  - "의도 고정"
  - (짧은 팝업) "LOCK"
- EXECUTE(집행):
  - "집행"
  - "실행"

> 권장: 실제 HUD에는 한국어 1개만 쓰고(예: "확정"), 개발/로그에서는 LOCK 용어 유지.
