# Obsidian Vault (Blood Contract)

이 폴더는 **Obsidian vault**다: Obsidian이 마크다운 파일들을 링크드 노트처럼 읽어주는 “폴더 기반 지식베이스”.

## 여는 방법
1) Obsidian 설치: https://obsidian.md
2) Obsidian → **Open folder as vault**
3) 이 폴더 선택: `blood_contract/docs/obsidian_vault/`

## 이 vault를 어떻게 쓰는가 (PoC)
- 프로젝트 네비게이션/대시보드/세션 로그를 여기서 관리한다.
- 진실의 원천(SSOT) 문서는 repo의 `docs/PACKETS/`에 유지한다.
- 이 vault에서는 PACKETS를 **복사하지 말고 링크로 연결**한다.
