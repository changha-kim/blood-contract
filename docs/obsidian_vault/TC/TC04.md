# TC04 — Charger 3초 명장면 재현성

## 목표(Goal)
- 10 attempts를 실행해서 성공률/실패 사유 분포를 기록한다.
- Week1 임시 기준: **≥ 3/10 성공**(또는 최소 1/5) + 실패 분류 가능

## 위치(Where)
- Godot → MainMenu → **TC04 (Spike Arena)**
- Scene: `res://scenes/levels/SpikeArena_GauntletLane.tscn`

## 시도 마킹 방법(핫키)
- **F5** = success + reset
- **F6** = fail + reset
  - Shift+F6 = control
  - Ctrl+F6 = readability
  - Alt+F6 = rules
  - (no modifier) = other
- **F7** = reset only (no attempt log)

## 증거(Evidence)
- Log file: `user://logs/run_*.jsonl`
- Event: `tc04_attempt` (run_id, room_id, attempt_id, success_bool, fail_reason_enum?, ts_msec)
