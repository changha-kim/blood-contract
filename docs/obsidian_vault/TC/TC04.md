# TC04 — Charger 3초 명장면 재현성

## Goal
- 10 attempts 실행해서 성공률/실패 사유 분포를 기록한다.
- Week1 기준 임시: **≥ 3/10 성공**(또는 최소 1/5) + 실패 분류 가능

## Where
- Godot → MainMenu → **TC04 (Spike Arena)**
- Scene: `res://scenes/levels/SpikeArena_GauntletLane.tscn`

## How to mark attempts (hotkeys)
- **F5** = success + reset
- **F6** = fail + reset
  - Shift+F6 = control
  - Ctrl+F6 = readability
  - Alt+F6 = rules
  - (no modifier) = other
- **F7** = reset only (no attempt log)
